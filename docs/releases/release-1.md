# Release 1 - Initial Playable Demo

Branch/commit reference: `origin/project-release-1`, commit `9d94181`

## Summary

Release 1 established the first playable File Not Found terminal demo. It included
the core domain models, static case data, validation helpers, command-line
interaction flow, and the first pass at engine behavior.

## Features Included

- Terminal-based case selection and gameplay loop.
- Case data for the investigation scenarios.
- Document unlocking through keyword searches.
- Evidence collection and final suspect submission.
- Validation logic for suspect and evidence correctness.

## Verification Status

Release 1 was the baseline implementation. Later Release 2 work added expanded
automated tests, model test coverage, documentation output, coverage reports,
and repository cleanup evidence.
